🚀 HOW TO BUILD:

  RECOMMENDED: Use IntelliJ IDEA (Easiest)
  ─────────────────────────────────────────
  1. Download IntelliJ IDEA Community (free)
  2. Open D:\blood-utilitys as project
  3. Wait for Gradle sync
  4. View → Tool Windows → Gradle
  5. Tasks → build → double-click build
  6. JAR created at: build/libs/blood-utilitys-1.0.0.jar

  ALTERNATIVE: Use VS Code
  ─────────────────────────
  1. Install "Gradle for Java" extension
  2. Open folder in VS Code
  3. Click Gradle icon → Tasks → build → play

  COMMAND LINE: (if Java 21 installed)
  ────────────────────────────────────
  cd D:\blood-utilitys
  gradle-8.5\bin\gradle.bat build

  ⚠️  NOTE: Your system has Java 25 (too new for Gradle 8.5)
      Use IntelliJ IDEA or VS Code for automatic compatibility handling
      See FINAL_BUILD_GUIDE.md for detailed solutions
