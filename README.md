# BLOOD UTILITYS

A clean, client-side PvP utility mod for Minecraft 1.21.1 with advanced HUD and hitbox rendering.

## Features

### Custom PvP HUD
- Armor durability percentage for each piece
- Totem count (inventory + offhand)
- FPS counter
- Ping display
- Active potion effects with timers
- Draggable HUD elements with edit mode

### Advanced Hitbox System
- Enhanced entity hitbox rendering
- Customizable colors, thickness, and transparency
- Toggle outline and/or fill rendering
- Only renders when appropriate

### Full Settings GUI
- Module toggles (HUD, Hitbox)
- HUD element repositioning
- Hitbox customization (color, thickness, alpha)
- Clean red and black themed interface

### Keybinds
- **Right Shift**: Open Settings GUI
- **H**: Toggle HUD
- **B**: Toggle Hitbox

All keybinds can be customized in Minecraft Controls menu.

## Installation

1. Download the mod JAR from releases
2. Place it in your `.minecraft/mods` folder
3. Launch Minecraft with Fabric Loader

## Configuration

Settings are saved in `config/blood-utilitys.json` and can be modified through the in-game GUI.

## Performance

This mod is optimized for performance with minimal FPS impact. It uses efficient rendering and only processes visible entities.
